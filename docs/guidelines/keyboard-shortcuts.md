# Cursor Keyboard Shortcuts - คีย์ลัดสำหรับ Windows และ Mac

คู่มือคีย์ลัดสำหรับใช้งาน Cursor AI ในโปรเจค PURA PMS

---

## 🎹 Keyboard Shortcuts หลัก

### AI Features

| ฟีเจอร์               | Mac     | Windows  | คำอธิบาย                                                       |
| --------------------- | ------- | -------- | -------------------------------------------------------------- |
| **Chat**              | `Cmd+L` | `Ctrl+L` | เปิด Chat เพื่อถามคำถาม, แตก Task, รีวิว Architecture          |
| **Composer**          | `Cmd+I` | `Ctrl+I` | เปิด Composer เพื่อเขียนโค้ด, สร้างไฟล์, แก้ไขหลายไฟล์พร้อมกัน |
| **Inline Edit**       | `Cmd+K` | `Ctrl+K` | แก้ไขโค้ดแบบ inline (ในบรรทัดเดียว)                            |
| **Accept Suggestion** | `Tab`   | `Tab`    | ยอมรับคำแนะนำจาก AI                                            |
| **Reject Suggestion** | `Esc`   | `Esc`    | ปฏิเสธคำแนะนำ                                                  |

### การใช้งานตามบทบาท

#### 🧑‍💼 Project Manager / Architect

- **ใช้:** Chat (`Ctrl+L` / `Cmd+L`)
- **เมื่อไหร่:** แตก Task, รีวิว Architecture, ถามคำถาม

#### 👨‍💻 Backend / Frontend Developer

- **ใช้:** Composer (`Ctrl+I` / `Cmd+I`)
- **เมื่อไหร่:** เขียนโค้ด, สร้างไฟล์, แก้ไขหลายไฟล์พร้อมกัน

#### 🕵️ QA Engineer

- **ใช้:** Composer (`Ctrl+I` / `Cmd+I`)
- **เมื่อไหร่:** เขียน Test, สร้างไฟล์ Test

## 💡 Best Practices

### 1. จำ Shortcuts หลัก

- **Chat:** `Ctrl+L` / `Cmd+L` - สำหรับถามคำถาม
- **Composer:** `Ctrl+I` / `Cmd+I` - สำหรับเขียนโค้ด

### 2. ใช้ให้ถูกสถานการณ์

- **Planning/Review:** ใช้ Chat
- **Coding:** ใช้ Composer
- **Quick Fix:** ใช้ Inline Edit

### 3. Customize Shortcuts

หากต้องการเปลี่ยน shortcuts:

1. ไปที่ Settings → Keyboard Shortcuts
2. ค้นหา "Cursor Chat" หรือ "Cursor Composer"
3. เปลี่ยนเป็น shortcuts ที่คุณต้องการ

## 🎯 สรุป

### Windows Users

- **Chat:** `Ctrl+L`
- **Composer:** `Ctrl+I`

### Mac Users

- **Chat:** `Cmd+L`
- **Composer:** `Cmd+I`

### หลักการจำ

- **L** = **L**isten (Chat - ฟังคำถาม)
- **I** = **I**mplement (Composer - เขียนโค้ด)
